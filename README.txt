En la presente rama se trabajan las promesas para las funciones de crear tarea y eliminar tarea.
Conclusiones:
1)	El uso del método then y async await a la postre refleja los mismos resultados en cuanto a funcionalidad
2)	Aunque la funcionalidad no cambie con los métodos, la limpieza del código si puesto que, al usar async await se realizar una tarea que, a pesar de ser asíncrona, la forma en la que se lee el código es similar a cuando se realiza una tarea síncrona 
3)	La facilidad de lectura del código con async await permite no generar el famoso callback hell y, además, que sea más fácil de leer para terceros.
Espero que estas conclusiones sean de tu agrado.
